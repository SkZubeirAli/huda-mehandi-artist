# Huda — Mehndi Artist Portfolio 🌿

A visually rich, single-file portfolio website for a professional mehndi (henna) artist.
Built with pure HTML, CSS, and JavaScript — no frameworks required.

## ✨ Features

- 🎨 SVG henna mandala loader (draws stroke-by-stroke, runs once per session)
- 🔤 Staggered letter-by-letter hero name animation
- 📜 Lenis smooth scroll + GSAP ScrollTrigger throughout
- 📌 Pinned horizontal scroll section with floating cone animation
- 📸 Auto-scrolling Instagram-style recent work slider (draggable)
- 🤝 Collaborations section with AOS fade-up card animations
- 🏆 Masonry awards & certificates gallery with hover reveal
- 📬 PHP contact form with dual-receiver emails + auto-reply to visitor
- 🔔 Top-right toast notification on form submit
- 📊 Scroll progress bar
- 🌓 Smart navbar (light/dark mode based on section background)
- 📱 Fully responsive (mobile hamburger menu)
- ⬆️ Back to top button

## 🛠 Tech Stack

HTML · CSS · JavaScript · GSAP 3 · Lenis · AOS · PHP (contact form)

## 🚀 Usage

Open `index.html` in a browser for the full frontend experience.  
For the contact form, deploy on a PHP server and update the email addresses in `contact.php`.
